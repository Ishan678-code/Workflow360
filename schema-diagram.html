Workflow360 – Database Schema

USER
Field NameData TypeConstraintsDescription
_id ObjectId Primary Key, Not Null Unique identifier for a user.
name varchar(255) Not Null Full name of the user.
email varchar(255) Not Null, Unique Email address used for login.
password varchar(255) Not Null Hashed password of the user.
role varchar(255) Not Null Role of the user (ADMIN / EMPLOYEE / MANAGER / FREELANCER).
isActive boolean Default: true Whether the user account is active.
lastLogoutdate Default: null Timestamp of the user's last logout.
EMPLOYEE
Field NameData TypeConstraintsDescription
_id ObjectId Primary Key, Not Null Unique identifier for an employee.
userId ObjectId Foreign Key, Not Null Reference to the User who is this employee.
employeeCode varchar(255) Not Null Company-assigned employee code.
department ObjectId Foreign Key Department the employee belongs to.
designation ObjectId Foreign Key Job role / designation of the employee.
manager ObjectId Foreign Key Reporting manager of the employee.
salary number Not Null Base salary of the employee.
joiningDate date Not Null Date the employee joined the company.
phone varchar(20) Not Null Contact phone number.
emergencyContact varchar(20) Emergency contact number.
employmentType varchar(255) Default: FULL_TIME Nature of employment (FULL_TIME / PART_TIME / CONTRACT / INTERN).
workMode varchar(255) Default: ONSITE Work arrangement mode (ONSITE / HYBRID / REMOTE).
officeHours.startvarchar(10) Default: "09:00" Scheduled office start time.
officeHours.end varchar(10) Default: "17:00" Scheduled office end time.
leaveBalance.annual number Default: 30 Annual leave days available.
leaveBalance.sick number Default: 12 Sick leave days available.
leaveBalance.casual number Default: 12 Casual leave days available.
DEPARTMENT
Field NameData TypeConstraintsDescription
_id ObjectId Primary Key, Not Null Unique identifier for a department.
code varchar(255) Uppercase, Trim Short department code (e.g. HR, ENG).
name varchar(255) Not Null, Unique Full name of the department.
description varchar(255) Description of the department.
head ObjectId Foreign Key User who heads this department.
parentDepartment ObjectId Foreign Key, Default: null Parent department (for hierarchical structure).
location varchar(255) Physical location of the department.

Field NameData TypeConstraintsDescription
_id ObjectId Primary Key, Not Null Unique identifier for a role.

levelvarchar(255) Seniority level (e.g. Senior, Junior).
FREELANCER
Field NameData TypeConstraintsDescription
_id ObjectId Primary Key, Not Null Unique identifier for a freelancer.
userId ObjectId Foreign Key, Not Null Reference to the User who is this freelancer.
hourlyRate number Not Null Billing rate per hour.
skills varchar(255) List of skills the freelancer has.
portfolioUrl varchar(255) Link to the freelancer's portfolio.
timezone varchar(255) Timezone of the freelancer.
PROJECT
Field NameData TypeConstraintsDescription
_id ObjectId Primary Key, Not Null Unique identifier for a project.
code varchar(255) Uppercase, Trim Short project code.
name varchar(255) Not Null Full project name.
description varchar(255) Project description.
clientName varchar(255) Name of the client for this project.
budget number Total allocated project budget.
startDate date Project start date.
deadline date Not Null Project deadline date.
status varchar(255) Default: ACTIVE Current status (ACTIVE / COMPLETED / ON_HOLD / CANCELLED).
manager ObjectId Foreign Key Project manager.
department ObjectId Foreign Key Department responsible for the project.
ownerManager ObjectId Foreign Key Project owner / sponsor.
priority varchar(255) Default: MEDIUM Priority level (LOW / MEDIUM / HIGH / URGENT).
estimatedHours number Estimated total hours for the project.
utilizationTargetnumber Default: 75 Target resource utilisation percentage.
requiredSkills varchar(255) Skills required to work on this project.
employees ObjectId[ ] Foreign Key Employees assigned to this project.
freelancers ObjectId[ ] Foreign Key Freelancers assigned to this project.
milestones varchar(255) List of project milestones.
TASK
Field NameData TypeConstraintsDescription
_id ObjectId Primary Key, Not Null Unique identifier for a task.
title varchar(255) Not Null Title / name of the task.
descriptionvarchar(255) Detailed description of the task.
project ObjectId Foreign Key, Not Null Project this task belongs to.
assignee ObjectId Foreign Key User assigned to complete the task.
createdBy ObjectId Foreign Key User who created the task.
status varchar(255) Default: TODO Task status (TODO / IN_PROGRESS / COMPLETED).
priority varchar(255) Default: MEDIUM Task priority (LOW / MEDIUM / HIGH / URGENT).
deadline date Deadline date for the task.
tags varchar(255) Labels / tags associated with the task.
ATTENDANCE
Field NameData TypeConstraintsDescription
_id ObjectId Primary Key, Not Null Unique identifier for an attendance record.
employee ObjectId Foreign Key, Not Null Employee this attendance record belongs to.
date date Not Null Date of the attendance record.
clockIn date Clock-in timestamp.
clockOut date Clock-out timestamp.
terminalId varchar(255) Terminal ID used for check-in.
officeStart varchar(10) Scheduled office start time for the day.
officeEnd varchar(10) Scheduled office end time for the day.
lateMinutes number Default: 0 Minutes arrived late.
earlyExitMinutes number Default: 0 Minutes of early departure.
overtimeHours number Default: 0 Overtime hours logged.
status varchar(255) Default: ON_TIME Attendance status (ON_TIME / LATE / EARLY_EXIT / COMPLETED).
LEAVE
Field NameData TypeConstraintsDescription
_id ObjectId Primary Key, Not Null Unique identifier for a leave request.
employee ObjectId Foreign Key, Not Null Employee requesting the leave.
type varchar(255) Not Null Category of leave (SICK / CASUAL / VACATION).
from date Not Null Leave start date.
to date Not Null Leave end date.
totalDays number Total number of leave days requested.
reason varchar(255) Reason provided for the leave.
status varchar(255) Default: PENDING Approval status (PENDING / APPROVED / REJECTED).
approvedBy ObjectId Foreign Key User who approved or rejected the leave.
comment varchar(255) Reviewer's comment on the leave.
approvalDepartment ObjectId Foreign Key Department responsible for approval.
PAYROLL
Field NameData TypeConstraintsDescription
_id ObjectId Primary Key, Not Null Unique identifier for a payroll record.
employee ObjectId Foreign Key, Not Null Employee this payroll belongs to.
month varchar(255) Not Null Payroll month (e.g. 2025-03).
payrollNumber varchar(255) Unique Auto-generated payroll reference number.
status varchar(255) Default: GENERATED Payment status (GENERATED / PAID).
grossSalary number Default: 0 Gross salary before deductions.
deductions number Default: 0 Total deductions applied.
netSalary number Default: 0 Net salary after deductions.
generatedBy ObjectId Foreign Key Admin who generated the payroll.
breakdown.totalDaysInMonth number Total calendar days in the payroll month.
breakdown.daysAttended number Number of days the employee attended.
breakdown.leaveDays number Approved leave days taken.
breakdown.lopDays number Loss-of-pay days.
breakdown.lopDeduction number Deduction amount for loss-of-pay days.
breakdown.taxDeduction number Tax deducted at source.
breakdown.lateDays number Number of days the employee was late.
breakdown.lateMinutes number Total late minutes accumulated.
breakdown.overtimeHours number Overtime hours logged in the month.
PERFORMANCE
Field NameData TypeConstraintsDescription
_id ObjectId Primary Key, Not Null Unique identifier for a performance review.
user ObjectId Foreign Key, Not Null Employee being reviewed.
reviewer ObjectId Foreign Key, Not Null Manager conducting the review.
rating number Not Null, Min: 1, Max: 5Performance rating on a 1–5 scale.
feedback varchar(255) Written feedback from the reviewer.
period varchar(255) Not Null Review period (e.g. Q1-2025).
TIMESHEET
Field NameData TypeConstraintsDescription
_id ObjectId Primary Key, Not Null Unique identifier for a timesheet entry.
freelancer ObjectId Foreign Key, Not Null Freelancer who logged the hours.
project ObjectId Foreign Key, Not Null Project the hours are logged against.
hours number Not Null Number of hours worked.
date date Not Null Date the work was performed.
description varchar(255) Description of work done.
qualityRating number Min: 1, Max: 5 Quality rating of the submitted work.
status varchar(255) Default: PENDING Approval status (PENDING / APPROVED / REJECTED).
INVOICE
Field NameData TypeConstraintsDescription
_id ObjectId Primary Key, Not Null Unique identifier for an invoice.

project ObjectId Foreign Key, Not Null Project the invoice is raised for.

amount number Not Null Total invoice amount.

dueDate date Payment due date.
status varchar(255) Default: PENDING Payment status (PENDING / PAID / CANCELLED).
